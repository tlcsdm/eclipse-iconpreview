Introduction

Icon files are directly showed in Package Explorer or other navigators.

This plugin supports the following image formats:

  • png
  • jpg/jpeg
  • gif
  • bmp
  • ico

Support/Bugs/Feature Requests

If you have some thoughts about IconPreview, be it a bug report a feature request or whatever, use issues section of the project's GitHub page. If the plug-in does not work properly in your environment or if you have an idea for a nice feature that would be suitable for this plug-in, just go ahead and tell me.
